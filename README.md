#### 리눅스 커맨드 라인 기초
- `pwd` : print working directory의 약자로 현재 작업 경로를 출력해주는 명령어
- `cd` : change directory의 약자로 현재 작업 경로를 변경할 때 쓰는 명령어
- `ls` : list의 약자로 현재 작업 경로에 있는 파일이나 디렉터리를 출력해주는 명령어
 - `ls -a`, `ls -l`, `ls -al`과 같은 옵션을 붙일 수 있음
